# **Downloads of subtitles from Hololive Youtube channels**  

Since yt seem to purge their proprietary srv3 format and nuking all user provided subtitles along the way, I used yt-dlp to back them up. 

- At the time I started this, some files may have been gone already and thus, could’nt be saved here
- Member content was not accessible and was skipped

## **Repository**

**Directory naming scheme:** Alias - Channel name

**Directory hierarchy**

```
channel
```
